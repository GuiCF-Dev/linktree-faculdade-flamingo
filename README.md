# linktree-faculdade-flamingo
Linktree realizando em sala de aula com o objetivo de ensinar o uso de html e css

🔗 Linktree - HTML e CSS
📖 Sobre o Projeto

Este projeto consiste na criação de uma página no estilo Linktree, desenvolvida durante as aulas da Faculdade Flamingo com o objetivo de ensinar e praticar os fundamentos de HTML e CSS.

A atividade tem como foco a construção de uma interface simples e organizada para centralizar links importantes em uma única página, permitindo a aplicação de conceitos essenciais de estruturação e estilização de páginas web.

🎯 Objetivos de Aprendizagem
Compreender a estrutura básica do HTML.
Aplicar estilos utilizando CSS.
Trabalhar com imagens e links.
Utilizar Flexbox para alinhamento de elementos.
Criar layouts simples e responsivos.
Desenvolver boas práticas de organização de código.
